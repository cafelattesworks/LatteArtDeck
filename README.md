# 🌟 LatteArt Deck

美しいアニメーションを備えた高精度タッチパッド（PTP）、マクロパッド、テンキー、フルキーボードを統合した次世代のカスタム入力デバイスです。ESP32とLovyanGFXをベースに構築されており、直感的なタッチ＆スワイプ操作と物理エンコーダーによるPCコントロールを実現します。

## ✨ 主な特徴 (Features)

* **マルチモード対応:** PTP（タッチパッド）、マクロパッド、テンキー、キーボード、設定画面の5つのモードをシームレスに切り替え可能。
* **美しいエフェクト:** 指の動きに連動する7種類の高品質なパーティクル/描画エフェクト（Luminous, Web, Gravityなど）。
* **スマートジェスチャー:** エッジスワイプによるモード切替や、macOS/Windows両対応のマルチタッチジェスチャー。
* **ロータリーエンコーダー連動:** スクロール、音量調整、Undo/Redoなど6種類のPC操作をダイヤルに割り当て可能。
* **ゴースト・スクリーンセーバー:** 放置時に自動で起動する美しい自律型アニメーション機能。

---

## 🔄 画面（モード）の切り替え方法

本デバイスは、「物理ボタン」または「画面のエッジ（端）スワイプ」で直感的にモードを切り替えることができます。

* **物理ボタン（BOOTボタン）**
  * 本体ボタンを押すたびに `タッチパッド` ➔ `マクロパッド` ➔ `テンキー` ➔ `キーボード` ➔ `設定画面` の順にループします。
* **エッジスワイプ（画面の端からスワイプ）**
  * **左右のエッジからスワイプ:** 前後のモード（タッチパッド〜キーボード）を切り替えます。
  * **上端から下へ:** どこからでも一発で**「設定画面」**を開きます。
  * **下端から上へ:** どこからでも一発で**「タッチパッド（PTP）」**に戻ります。

## 👆 各モードの機能と使い方

### ① タッチパッドモード (PTP)
PCの高精度タッチパッドとして機能します。
* **エフェクト変更:** 画面の「右上隅」をタップすると、7種類のエフェクトが順番に切り替わります。
* **スクリーンセーバー:** 12秒間無操作でいると、自動的にゴーストタッチアニメーションが起動します（画面に触れると即座に復帰）。
* **ジェスチャー操作 (OS連動):**
  * **Windows:** Windows標準の高精度タッチパッド（PTP）として動作。OS側でジェスチャーをカスタマイズ可能です。
  * **macOS:** 以下の専用ジェスチャーが自動適用されます。
    * **1本指:** カーソル移動
    * **2本指:** タップで右クリック、上下でスクロール、ピンチでズーム、左右で戻る/進む
    * **3本指:** 縦でMission Control、横でアプリ切り替え（Cmd+Tab）
    * **4本指:** 縦でデスクトップ表示、横でデスクトップ間の移動

### ② マクロパッドモード (12キー)
コピー、ペースト、Undo、保存などのショートカットを配置した画面です。
* ※シリアル通信経由（Webアプリ等）で、各ボタンのラベルやマクロ動作、長文入力などを自由にカスタマイズ可能です。

### ③ テンキーモード (17キー)
数値入力に特化し、バックスペースやエンターキーを備えたエクセル等に最適な画面です。

### ④ キーボードモード (34キー)
フル機能のスクリーンキーボードです。
* **ページ切り替え:** 「?123」「#+=」「ABC」ボタンで、アルファベット / 数字・基本記号 / マニアックな記号・顔文字の3ページを切り替えられます。
* **スマートOS対応:** 設定で「macOS」を選択中、Ctrlキーは自動的に「Cmd」キーとして機能します。
* **長押し対応:** ShiftやCtrl等の修飾キーは、タップでホールド、再タップで解除されます。

## ⚙️ 設定画面 (Settings)

設定画面から以下の項目を変更できます。設定値は本体に自動保存されます。

| 設定項目 | 説明 | 選択肢 |
| :--- | :--- | :--- |
| **TouchPad Effect** | タッチパッドのエフェクト | 1.Luminous ～ 7.Flower (全7種) |
| **Brightness** | 画面の明るさ | 20%, 40%, 60%, 80%, MAX |
| **Auto Sleep** | 画面自動消灯までの時間 | Never (なし), 1 Min, 5 Min, 15 Min |
| **USB Cable** | ケーブルを出す向き(画面の反転) | Left (標準), Right (180度反転) |
| **Encoder Dial** | ダイヤルを回した時の動作 | 下記「エンコーダー操作」を参照 |
| **OS Mode** | 接続先のパソコンのOS | Windows, macOS |

*(※スリープ復帰時の最初のタッチは「画面点灯用」として処理され、誤動作を防ぐ安全設計になっています)*

---

## 🎛️ エンコーダーダイヤル操作

本体の回転ダイヤルには、設定画面の「Encoder Dial」から以下の6つの役割を割り当てられます。

1. **Scroll:** 画面の上下スクロール（↑/↓キー）
2. **Timeline:** タイムライン移動（←/→キー）
3. **Brush Size:** ブラシサイズ変更（ `[` と `]` ）
4. **Undo/Redo:** 取り消し/やり直し（Ctrl+Z / Ctrl+Y）
5. **Volume:** パソコンのシステム音量調整
6. **Brightness:** パソコンの画面の明るさ調整
